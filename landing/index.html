Frequently Asked Questions

Everything you need to know about RETRO.

Does RETRO require internet?

Partial. The sensors work fully offline. Some OSINT sources (Shodan, VirusTotal, AbuseIPDB) need internet. You can use RETRO with only local DNS/WHOIS for full offline mode.

Is it free?

100% free. No subscriptions, no cloud, no hidden costs. You just need Python and optionally Ollama for LLM features.

Do I need API keys?

Only if you want to use Shodan, VirusTotal, or AbuseIPDB — all offer free tiers. The core features (honeypots, DNS, WHOIS, GeoIP) work without keys.

Is it legal?

Yes. RETRO operates on your own infrastructure. Honeypots capture data from attackers targeting you. OSINT uses public sources. Countermeasures act on your own systems.

What operating systems does it support?

Linux (recommended). The firewall features require iptables/nftables. The Python backend works on any OS.
